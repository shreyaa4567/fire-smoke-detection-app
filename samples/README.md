# Quick Test Samples

Add sample images here for one-click testing. 
Name them descriptively — the filename becomes the button label.

Examples:
- `wildfire.jpg`       → 🔥 Wildfire
- `industrial_smoke.jpg` → 🏭 Industrial Smoke
- `forest_fire.jpg`    → 🌲 Forest Fire
- `safe_scene.jpg`     → ✅ Safe Scene

Supported formats: `.jpg`, `.jpeg`, `.png`, `.bmp`, `.webp`
