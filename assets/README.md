# Assets

This folder contains screenshots and diagrams for the README.

## Required Screenshots

Capture these screenshots from the running app (`streamlit run appnew.py`):

| Filename | Description | How to Capture |
|----------|-------------|----------------|
| `dashboard.png` | Full dashboard overview showing the header and mode selector | Open the app, take a full-page screenshot |
| `detection.png` | Image detection results showing original vs detection side-by-side | Upload a fire/smoke image, screenshot the results |
| `demo-mode.png` | Demo Mode running with status indicator and timeline | Start Demo Mode, screenshot while cycling |
| `video-processing.png` | Video upload results with annotated video and stats | Process a short video, screenshot the results |
| `quick-test.png` | Quick Test Samples buttons with detection output | Click a sample, screenshot the results |
| `webcam.png` | Webcam capture mode | Capture a photo, screenshot the output |

## Tips for Great Screenshots

1. Use a **1920×1080** browser window for consistent sizing
2. Use **Chrome DevTools** → Device Toolbar for clean captures
3. Ensure the **dark theme renders fully** before capturing
4. Crop to remove browser chrome (address bar, tabs)
5. Consider using a tool like **ShareX** or **Snipping Tool** for clean captures
